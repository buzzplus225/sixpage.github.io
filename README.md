# Page Six RSS Scraper - Traduction Automatique

## 📋 Description
Ce projet scrape automatiquement les articles de Page Six et les traduit en français en utilisant le modèle Helsinki-NLP/opus-mt-en-fr.

## 🚀 Fonctionnalités
- Scraping automatique toutes les 4 heures
- Traduction anglais → français
- Génération de flux RSS
- Hébergement sur GitHub Pages
- Interface de monitoring

## 🔧 Configuration requise
- Python 3.10+
- GitHub Actions
- 2GB RAM minimum

## 📦 Installation

```bash
# Cloner le dépôt
git clone https://github.com/buzzplus225/sixpage.github.io.git
cd legitoi.github.io

# Installer les dépendances
pip install -r requirements.txt

# Exécuter manuellement
python scraper_traducteur.py
